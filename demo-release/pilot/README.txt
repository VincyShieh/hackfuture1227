Space Journey - 单文件独立运行版本
====================================

✅ 真正支持双击运行的版本！
✅ 不需要任何服务器！
✅ 可以直接嵌入到其他 HTML 项目！


使用方法：
--------

方法 1：直接双击运行
  直接双击 game.html 文件即可在浏览器中运行游戏！
  不需要任何服务器，不需要 Live Server！


方法 2：集成到其他 HTML 项目
  将 game.html 的内容复制到你的项目中即可。
  
  或者手动集成：
  1. 复制 <style> 标签的内容到你的 CSS 文件或 <head> 中
  2. 复制 <script> 标签的内容到你的 JS 文件或 <body> 中
  3. 确保 <div id="root"></div> 存在


为什么这个版本可以双击运行？
------------------------------

之前的问题：
- Vite 默认使用 ES 模块（type="module"），浏览器禁止从 file:// 协议加载
- 虽然移除了 miaoda 插件，但 React 和其他库在模块化格式下仍有限制

现在的解决方案：
- 使用 IIFE（立即执行函数）格式打包
- 将所有代码内联到一个 HTML 文件中
- 完全不依赖外部模块加载


文件说明：
--------

- game.html        ⭐ 主要文件！双击这个文件运行游戏
- game.js          游戏主程序（已内联到 game.html）
- style.css        样式文件（已内联到 game.html）
- favicon.png      网站图标


性能说明：
--------

game.html 文件大小约 346 KB，包含：
- 游戏代码（约 328 KB）
- 样式（约 18 KB）
- HTML 结构

这是一个完全独立的版本，加载速度很快，不需要额外的网络请求。


注意：
-----

1. 如果在其他项目中使用，确保页面中只有一个 <div id="root"></div>
2. 如果项目中已经有 React，可能会产生冲突，请确保 ID 不重复
3. 推荐在现代浏览器中使用（Chrome, Firefox, Edge, Safari 等）


问题排查：
--------

如果游戏无法运行：
1. 确保使用现代浏览器（Chrome 90+, Firefox 88+, Edge 90+, Safari 14+）
2. 检查浏览器控制台是否有错误信息（按 F12 打开）
3. 确保文件没有被安全软件拦截


技术支持：
--------

这是一个真正的单文件版本，应该可以直接双击运行。
如果还有问题，请检查浏览器控制台的错误信息。
