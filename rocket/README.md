# 小火箭手 🚀

一款专为6-12岁儿童设计的火箭拼装和发射互动游戏，通过拖拽装配火箭部件，体验航天探索的乐趣。

## ✨ 游戏特色

- 🔧 **自由拼装**：拖拽各种火箭部件（整流罩、箭体、尾翼、助推器）进行组装
- 🎬 **精彩动画**：8秒完整发射动画，包含倒计时、点火、助推器分离、整流罩打开等阶段
- 🌟 **太空任务**：随机生成6种不同的太空探索任务故事
- 🎵 **音效系统**：使用Web Audio API生成的拖拽、放置、发射音效
- 📤 **分享功能**：通过URL参数分享你的火箭设计
- 💾 **自动保存**：使用localStorage自动保存最近的火箭配置
- 📱 **响应式设计**：适配平板和桌面设备

## 🎮 游戏玩法

1. **首页**：点击"开始拼装"按钮进入装配车间
2. **装配车间**：
   - 从左侧部件列表拖拽部件到右侧装配区
   - 必须装配：整流罩、箭体、尾翼（助推器可选）
   - 点击已装配部件可以移除
   - 完成后点击"发射火箭"按钮
3. **发射动画**：观看8秒完整的火箭发射过程
4. **任务报告**：
   - 查看随机分配的太空任务
   - 查看你的火箭配置
   - 点击"分享火箭"复制链接分享给朋友
   - 点击"再次拼装"返回装配车间

## 🎨 设计风格

- **配色方案**：宇宙蓝（#1a3648）+ 活力橙（#ff6b35）
- **视觉风格**：圆角设计、柔和阴影、SVG几何图标
- **动画效果**：浮动、震动、粒子、闪烁等多种CSS动画

## 🛠️ 技术栈

- **框架**：React 18 + TypeScript
- **样式**：Tailwind CSS + 自定义CSS动画
- **构建工具**：Vite
- **音效**：Web Audio API
- **存储**：localStorage + URL参数

## 🎯 游戏功能详解

### 拖拽系统
- 使用HTML5 Drag and Drop API实现
- 拖拽时显示半透明预览
- 放置时播放音效反馈

### 发射动画时间线
- 0-3秒：倒计时 + 点火发射
- 3-5秒：助推器分离（如果有）
- 5-8秒：整流罩打开
- 8秒+：进入轨道，显示成功信息

### 任务系统
- 6种预设任务：国际空间站补给、月球探测、卫星部署、火星探索、太空望远镜、气象卫星
- 每次发射随机选择一个任务
- 每个任务包含标题、描述和成就徽章

### 分享功能
- 将火箭配置编码为URL参数
- 一键复制分享链接
- 访问链接自动还原火箭配置
