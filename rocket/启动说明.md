# 🚀 小火箭手 - 启动指南

## 快速启动（推荐）

### 方法一：双击运行（最简单）

1. **Windows 用户**：双击 `start.bat` 文件
2. **PowerShell 用户**：在 PowerShell 中运行 `.\start.ps1`

脚本会自动：
- 检查并安装 pnpm（如果需要）
- 安装项目依赖
- 启动开发服务器
- 自动在浏览器中打开游戏

---

## 手动启动

### 步骤 1：安装 pnpm

```powershell
npm install -g pnpm
```

### 步骤 2：安装项目依赖

```powershell
cd f:\00hackthon\rocket
pnpm install
```

### 步骤 3：启动开发服务器

```powershell
pnpm run dev
```

游戏将在浏览器中自动打开！

## 游戏访问地址

启动成功后，游戏将在以下地址运行：

- **开发服务器**：http://localhost:5173
- **如果端口被占用**：Vite 会自动选择其他端口（如 5174, 5175...）

## 常见问题

### Q: 提示 "pnpm 不是内部或外部命令"
**A:** 需要先安装 pnpm：
```powershell
npm install -g pnpm
```

### Q: 依赖安装很慢
**A:** 这是因为项目有很多依赖，首次安装需要 3-5 分钟，请耐心等待。

### Q: 浏览器无法打开
**A:** 手动在浏览器中访问 `http://localhost:5173`

### Q: PowerShell 提示"无法加载脚本"
**A:** 需要先设置执行策略（仅限本次）：
```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
.\start.ps1
```
