# 使用指南 📖

## 快速开始

### 1. 安装依赖
```bash
pnpm install
```

### 2. 启动开发服务器
```bash
pnpm run dev
```

游戏将在浏览器中自动打开，通常地址为 `http://localhost:5173`

### 3. 构建生产版本
```bash
pnpm run build
```

构建产物将生成在 `dist` 目录中。

## 游戏操作指南

### 首页
1. 阅读游戏介绍
2. 点击 **"🚀 开始拼装"** 按钮进入装配车间

### 装配车间

#### 拖拽部件
1. 在左侧找到你想要的部件
2. 按住鼠标左键拖拽部件
3. 将部件拖到右侧的装配区
4. 松开鼠标完成放置
5. 听到"咚"的一声表示放置成功

#### 部件类型
- **整流罩**（必须）：火箭顶部的锥形部分，保护载荷
- **箭体**（必须）：火箭的主体部分，包含燃料和设备
- **尾翼**（必须）：火箭底部的翼片，保持飞行稳定
- **助推器**（可选）：提供额外推力，可以不装

#### 移除部件
- 点击装配区下方的部件标签（如"整流罩 ✕"）即可移除

#### 发射火箭
1. 确保已装配：整流罩、箭体、尾翼（三个必须部件）
2. 点击右上角的 **"🚀 发射火箭"** 按钮
3. 按钮会在部件齐全后变为橙色可点击状态

### 发射动画

观看8秒精彩的火箭发射过程：

- **0-3秒**：倒计时 3、2、1
- **3秒**：点火！火箭开始升空
- **3-5秒**：助推器分离（如果装了助推器）
- **5-8秒**：整流罩打开
- **8秒后**：成功进入轨道！

### 任务报告

#### 查看任务
- 系统会随机分配一个太空任务
- 阅读任务描述和获得的成就

#### 分享火箭
1. 点击 **"📤 分享火箭"** 按钮
2. 链接会自动复制到剪贴板
3. 将链接发送给朋友
4. 朋友打开链接后会看到你的火箭设计

#### 继续游戏
- 点击 **"🔧 再次拼装"** 返回装配车间
- 你的上一次配置会被保存

## 高级功能

### 自动保存
- 游戏会自动保存你最近的火箭配置
- 刷新页面后配置会自动恢复
- 数据保存在浏览器的localStorage中

### URL分享
- 分享链接包含完整的火箭配置信息
- 格式：`?body=颜色&nosecone=颜色&fins=颜色&booster=颜色`
- 打开链接会自动还原火箭设计

### 音效控制
- 游戏包含4种音效：拖拽、放置、发射、成功
- 音效使用Web Audio API实时生成
- 首次点击页面后音效会自动启用

## 浏览器兼容性

推荐使用以下浏览器：
- ✅ Chrome 90+
- ✅ Firefox 88+
- ✅ Safari 14+
- ✅ Edge 90+

需要支持的功能：
- HTML5 Drag and Drop API
- Web Audio API
- localStorage
- Clipboard API

## 故障排除

### 音效不播放
- 确保已点击过页面（浏览器安全策略要求）
- 检查浏览器是否静音
- 尝试刷新页面

### 拖拽不工作
- 确保使用鼠标操作（触摸屏可能不支持）
- 检查浏览器是否支持Drag and Drop API
- 尝试使用Chrome或Firefox浏览器

### 配置未保存
- 检查浏览器是否允许localStorage
- 确保未使用无痕/隐私模式
- 清除浏览器缓存后重试

### 分享链接无效
- 确保完整复制了整个URL
- 检查浏览器是否支持Clipboard API
- 手动复制地址栏中的URL

## 开发相关

### 项目结构
```
src/
├── components/     # React组件
├── types/          # TypeScript类型
├── data/           # 游戏数据
├── utils/          # 工具函数
├── App.tsx         # 主应用
└── index.css       # 样式文件
```

### 修改部件
编辑 `src/data/gameData.ts` 中的 `ROCKET_PARTS` 数组：
```typescript
{
  id: 'body-3',
  type: 'body',
  name: '你的箭体名称',
  color: '#你的颜色',
  description: '描述文字'
}
```

### 添加任务
编辑 `src/data/gameData.ts` 中的 `MISSION_STORIES` 数组：
```typescript
{
  title: '任务标题',
  description: '任务描述',
  achievement: '成就名称'
}
```

### 调整动画
编辑 `src/components/LaunchAnimation.tsx` 中的时间参数：
```typescript
setTimeout(() => setPhase('booster-sep'), 3000); // 3秒后助推器分离
```

2. 是否使用了推荐的浏览器版本
3. 网络连接是否正常
